# Bundled fonts

These caption fonts are bundled so burned-in captions work without installing
anything. All are licensed under the **SIL Open Font License 1.1** (OFL), which
permits redistribution.

| File | Family | Source |
|------|--------|--------|
| Anton-Regular.ttf | Anton | https://fonts.google.com/specimen/Anton |
| BebasNeue-Regular.ttf | Bebas Neue | https://fonts.google.com/specimen/Bebas+Neue |
| Montserrat-Variable.ttf | Montserrat | https://fonts.google.com/specimen/Montserrat |

System fonts (Arial Black, Impact) are not bundled — they're found on the OS.
