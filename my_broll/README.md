# Your own B-roll

Drop images, GIFs, and videos in this folder, **named by what they show**, and
the editor will use them instead of random stock when the words match.

- Name by keywords: `desk-lamp.jpg`, `mechanical_keyboard.mp4`, `amazon-box.gif`,
  `kitchen timer.mp4`. Words are matched against what you say.
- Any aspect ratio is fine (it's cropped to fit). GIFs animate and loop.
- Supported: .jpg .jpeg .png .webp .bmp .gif .mp4 .mov .webm .m4v

Turn it on: in the app set **B-roll source = "My media only"** (or "My media, then
stock"). CLI: `--broll --broll-source mine` (or `mix`).
